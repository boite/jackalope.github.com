Content Repository for PHP

Jackalope is an open source PHP implementation of the PHPCR API, which is a PHP adaption of the Java Content Repository (JCR) standard, an open API specification defined in JSR-170/283.

Jackalope's test-driven development aims to provide full client functionality for Apache Jackrabbit, the JCR reference implementation as well as being a base to build a full PHP Content Repository with no Java backend involved.
